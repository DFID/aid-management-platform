Text

We will use "font-family: Arial, Helvetica, sans-serif;".
Default font size is 16px
You can find common font colours in the colour palette.

Headings

  • use Arial Bold
  • use sentence case for headings
  • use headings consistently to create a clear hierarchy

h1 A 32px Bold heading

h2 A 26px Bold heading

h3 A 20px Bold heading

h4 A 16px Bold heading

    
<h1>h1 A <em class="highlight">32px Bold</em> heading</h1>

    <h2>h2 A <em class="highlight">26px Bold</em> heading</h2>

    <h3>h3 A <em class="highlight">20px Bold</em> heading</h3>

    <h4>h4 A <em class="highlight">16px Bold</em> heading</h4>

Lead paragraph and body copy

A 24px lead paragraph.

A 19px large font paragraph.

A 16px normal font paragraph.

    
<div class="text">
        <p class="lead">A <em class="highlight">24px</em> lead paragraph.</p>

        <p class="font-large">A <em class="highlight">19px</em> large font paragraph.</p>

        <p class="font-normal">A <em class="highlight">16px</em> normal font paragraph.</p>
    </div>

Body copy

  • use Arial, Helvetica, sans-serif
  • avoid using bold and italics
  • use 16px for body copy – no need to reduce for smaller screens
  • use smaller sizes only if there's a user need (eg 14px, 12px)
  • make sure lines of text don't exceed 75 characters, as they become harder to read beyond that point
  • for example, on this page – text blocks are two-thirds of the page width
  • when the screen width is below 640px - text becomes 100% of the page width

A 19px large font paragraph. Maecenas sed diam eget risus varius blandit sit amet non magna. Donec ullamcorper nulla non metus auctor fringilla.

A 16px normal paragraph. Maecenas sed diam eget risus varius blandit sit amet non magna. Donec ullamcorper nulla non metus auctor fringilla.

Links

  • links within body copy should be blue and underlined
  • links without surrounding text should not have a full stop at the end
  • link colours can be found in the colour palette

A 16px link without surrounding text

A 16px body copy link. Fusce dapibus, tellus ac cursus commodo, tortor mauris condimentum nibh, ut fermentum massa justo sit amet risus.

A 16px body copy external link. Fusce dapibus, tellus ac cursus commodo, tortor mauris condimentum nibh, ut fermentum massa justo sit amet risus.

The default button colour background is '$dfid-green' as declared in the SASS '_colours' file
Alternative colours can be used as shown below
The pseudo states (hover / active / visited) are automatically rendered 

    
<button>Save</button>
    <button class="red">Reject</button>
    <button class="bluelightnight">Interact</button>
    <button class="punchoutgrey">Leave System</button>

The SASS to show the different coloured buttons

    

.button{
  @include button;
}

  @include button($dfid-green);
  &.blue {
    @include button($dfid-blue);
  }
  &.red {
    @include button($dfid-red);
  }
  &.purple {
    @include button($dfid-purple);
  }
   &.bluelightnight {

  }
 &.punchoutgrey {

the Javascript for Accordions..

    
// Simple Accordion script
    $(document).ready(function($) {
        $(".accordion-content").slideUp('fast');
        $('#accordion').find('.accordion-toggle').click(function(){

        //Expand or collapse this panel
        $(this).next().slideToggle('medium');
        $(this).toggleClass('visible');
        $(".accordion-toggle").not(this).removeClass('visible');

        //Hide the other panels
        $(".accordion-content").not($(this).next()).slideUp('medium');

        });
    });
    // Simple Accordion script 1 - copied for quickness to allow multi-accordions on page
    $(document).ready(function($) {
        $(".accordion-1-content").slideUp('fast');
        $('#accordion-1').find('.accordion-1-toggle').click(function(){

        //Expand or collapse this panel
        $(this).next().slideToggle('medium');
        $(this).toggleClass('visible');
        $(".accordion-1-toggle").not(this).removeClass('visible');

        //Hide the other panels
        $(".accordion-1-content").not($(this).next()).slideUp('slow');

        });
    });

You can also add a class tag within the slideshow wrapper tag to be able to control dimensions, drop shadows, background colours etc.
